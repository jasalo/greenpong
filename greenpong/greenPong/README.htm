Notas generales

GreenPong es un proyecto de estudiantes de ingenieria de la universidad de los Andes en Bogota, Colombia. Se ofrece como código libre y abierto sin fines comerciales.

Configuracion para Ant en Linux

En linux se debe agregar una linea al archivo .bashrc del home para definir una variable de entorno. Una vez hecho esto se debe reiniciar el etorno grafico y no se requiere repetir el proceso en adelante. El comando para agregar la linea en el home del usuario es:

echo export JAVA_HOME="/usr/lib/j2sdk1.5-sun/" >> ~/.bashrc

Configuracion para Ant en Windows

Se debe saber en que sitio esta la carpeta del SDK y dando click derecho MI PC Definir la variable de entorno JAVA_HOME y darle la ruta al SDK.
